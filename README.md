# Mirelli Pimentel — Perita Judicial

Landing page institucional (site estático, arquivo único, funciona offline).

## Como publicar
1. Suba esta pasta para um repositório no GitHub.
2. No Vercel, importe o repositório (vercel.com/new) e clique em Deploy.
3. Não há build: é HTML estático. O Vercel serve o `index.html` automaticamente.

## Atualizar o site
Substitua o `index.html` por uma versão nova e faça commit/push.
O Vercel republica sozinho.
